B — Pipeline

Left-to-right flow: sources → extract → output. Shows the user exactly what the tool does. Functional, honest, no decoration.

For

  • Makes the process transparent
  • Feels like a tool, not an app
  • Expert toggles fit naturally

Against

  • May feel cold to non-technical users

C — Minimal ★ recommended

Almost nothing on screen. One number. One button. Everything else is one click away. Feels like a serious instrument, not a product. The restraint signals confidence.

For

  • Memorable, distinctive
  • Matches the product's tone — cold, precise
  • Non-technical users aren't overwhelmed
  • Expert mode accessible but not default

Against

  • Needs great onboarding to set up sources

These aren't mutually exclusive — C's shell can contain B's pipeline inside it. Which outer face feels right for EXHAUST?
